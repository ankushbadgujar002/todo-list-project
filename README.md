# Todo List Application

## 📌 Project Description

This is a simple and interactive Todo List application built using React.js. It allows users to add, delete, and manage daily tasks efficiently.

## 🚀 Features

- Add new tasks
- Delete tasks
- Mark tasks as complete
- Dynamic UI updates
- State management using React Hooks

## 🛠️ Technologies Used

- React.js
- JavaScript
- HTML
- CSS

## 📂 Project Structure

src/
components/
App.js
index.js

## ▶️ How to Run

1. Clone the repository
2. Run npm install
3. Run npm start
